HOW TU USE GIT

Description Commandes
Ajouter tout les éléments git add *
Commit qui permet de le stocker en local
(commenter aussi les rajout et autres) 		git commit -m "nomducommit"
Envoyer ses commits vers le dépôt distant git push
Envoyer vers le dépôt distant a une branche spécifique git push origin MA_BRANCH
Mettre à jour le dépôt local git pull
Mettre à jour le dépôt local d'une branch spécifique git pull origin MA_BRANCH
Créer une nouvelle branche ou naviguer entre elle git checkout
Créer une nouvelle branche ou naviguer entre elle git checkout
Voir toute les commits d'un repertoire git log
Retourne la liste des tags disponible git tag
Initialise un nouveau répertoire git init

Importer un projet:

cd chemin/vers/mon_dossier
git add *
git commit -m "Initial commit"
git push origin main

Initialiser un projet:

cd chemin/vers/mon_dossier
echo "# MON_PROJET" >> README.md
git init
git add README.md
git commit -m "Initial commit"
git remote add origin ....git
git push -u origin master
